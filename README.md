# 🎬 CineBai

**CineBai** is a full-stack **AI-powered movie and TV show streaming platform** built with **React, Node.js, Express, and MongoDB**. Powered by the **TMDB API** and **Gemini AI**,CineBai delivers AI-Powered prompt-based discovery streaming experience for movie/tv lovers. 

---

## 🚀 Features
🤖 **Gemini AI Recommendations** – Get personalized suggestions based on your preferences or mood  
🔐 **Secure Authentication** – Sign up and log in using JWT-based authentication with sign in via gmail

🔍 **Smart Search** – Search for movies and TV shows in real time  
🎥 **Watch & Explore** – Access detailed information,and related titles  
❤️ **Favorites** – Add and manage your favorite movies and shows  
🕒 **Watch Later** – Save content for future viewing  
🔄 **Session Persistence** – Stay logged in seamlessly using localStorage  
📡 **Live TMDB Data** – Fetch up-to-date movie and TV show details  

---

## 🧠 Tech Stack

**Frontend:** React, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**APIs:** TMDB API, Gemini AI API  
**Authentication:** JSON Web Tokens (JWT)
